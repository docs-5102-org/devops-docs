# cal命令-显示系统月历与日期

Cal命令用于在终端中显示日历，是Linux系统中一个简单实用的命令行工具。

## 命令格式

```bash
cal [选项] [月份] [年份]
```

## 基本用法

```bash
# 显示当前月份的日历
cal

# 显示整年的日历
cal -y

# 显示指定年份的日历
cal 2023

# 显示指定月份和年份的日历
cal 12 2023
```

## 常用选项

- `-1` 仅显示当前月份（默认）
- `-3` 显示上个月、当前月和下个月
- `-y` 显示当前年份的日历
- `-j` 显示一年中的第几天（儒略日）
- `-m` 以星期一为一周的第一天（默认为星期日）
